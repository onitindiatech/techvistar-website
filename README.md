# 🚀 TechVistar Website

A modern, scalable, and production-ready company website built using **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

The project follows a clean architecture where UI components and business content are completely separated, making the application easy to maintain, scale, and extend in the future.

---

TechVistar is a technology-first company website designed with a modern SaaS-inspired user experience.

The application focuses on:

# 📂 Project Structure

```text
src/
│
├── components/      # UI Components
├── data/            # Dynamic Website Content
├── hooks/           # Custom React Hooks
├── lib/             # Utilities & Global Constants
├── pages/           # Route Pages
├── App.tsx
└── main.tsx
```

---

# ⚙️ Architecture

The project follows **Separation of Concerns**.

- UI Components only render the interface.
- Business content is stored inside the `src/data` directory.
- Shared utilities remain inside `src/lib`.
- Pages compose reusable components.
- Dynamic rendering replaces hardcoded content.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Adityachoubey26/techvistar-website.git
```

Move into the project

```bash
cd techvistar-website
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```
